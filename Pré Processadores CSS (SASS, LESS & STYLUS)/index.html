Pré Processadores CSS - Benefícios: Programação dentro do CSS • Capacidade de realizar loop • Centralização de código • Suporte a imports • Código limpo, modularizado, fácil manutenção, reaproveitável Exemplos de pré processadores: SASS: Desenvolvido em Ruby e tem código fonte aberto (tem suporte a funções); LESS: Não repita você mesmo = impede repetir qualquer código CSS - Desenvolvido em Ruby e foi migrado para JS (usa Mixins) ; STYLUS: Baseado em JS utilizando Node (tem suporte a funções). Funcionalidades dos PRÉ PROCESSADORES: Variaveis: não tem tipo, podem armazenar valores simples etc...; Mixins: ajuda no reaproveitamento do código e pode ser chamado por diversos seletores e permitem passar parametros; Funções: retorna um unico valor (seja numerico ou uma propriedade); Extend: propriedades que se repetem (sem preocupar em separar por virgula igual no CSS puro); No final, todo pré processador gera um código CSS puro que é interpretado pelo navegador
